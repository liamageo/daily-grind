Similar to our previous web page, we continue to practice with JavaScript. In this webpage we are demonstrating JavaScript's "Date" object, conditionals like a "switch" block, and linking URL parameters. By using the date object, we are able to change the content of the page based on the time, day or date of the week. By selecting any day of the week, the end-user is able to see the changes to the following: images, day of the week, beverage, and background color. End-users are also able to open the web page and see the correct drink for that specific day. We also created a function containing parameters and a switch statement. The main function parameter used was the word, "coffee", which allowed us to bind the elements of our string together. The "switch statement" allows as user to perform different actions based on different conditions. This, along with the date object, is what allows us to rotate the content by the day of the week.
